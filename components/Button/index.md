# Button 按钮

按钮用于开始一个即时操作。

## 基础用法

使用 type、plain、round 和 circle 来定义按钮的样式

## API

| 参数     | 说明             | 类型      | 默认值    | 版本 |
| -------- | ---------------- | --------- | --------- | ---- |
| action   | 开关状态         | `boolean` | `true`    |      |
| onColor  | 开启状态容器颜色 | `string`  | `#409eff` |      |
| offColor | 关闭状态容器颜色 | `string`  | `#dcdfe6` |      |
